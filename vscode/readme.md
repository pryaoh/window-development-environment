## 단축키 정리

- *Ctrl +*: 화면 확대
- *Ctrl -*: 화면 축소

- *Ctrl f*: 파일 내 문자열 검색
- *Ctrl p*: 파일 검색

- *Ctrl b*: 사이드바 전환
- *Ctrl \\*: 편집창 나누기
